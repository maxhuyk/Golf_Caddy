# Ingeniería Mecánica y Diseño Industrial

Este módulo contiene los entregables para la fabricación de los housings y accesorios de carga del sistema.

## 📂 Contenido Técnico
* **Planos Técnicos (PDF):** Planimetría de ensamble y dimensiones nominales.
* **Modelos de Producción (.STL):** Archivos optimizados para impresión 3D (FDM/SLA).
* **Archivos de Diseño (.PAR):** Modelos nativos (Solid Edge / CAD) para modificaciones de ingeniería o matricería.

## 🔩 Recomendaciones de Manufactura
* **Housings Exteriores:** Se recomienda el uso de **PETG ** o **ASA** debido a su resistencia mecánica y estabilidad ante la radiación UV.
* **Estación de Carga:** El diseño `soporte_carga.par`
* está optimizado para integrar bobinas de inducción inalámbrica estándar, permitiendo la recarga del TAG sin contacto metálico expuesto.
